    文献检索

    跨 OpenAlex / Crossref / arXiv 检索论文与资料。普通检索可翻页浏览、存入文库；智能检索会多步检索并生成一份简要检索报告。

    智能检索：输入一个方向或明确需求（如“帮我找近3-5年，大语言模型智能体（LLM Agent）在自动化文献综述中应用强相关的20篇文献，15篇中文，5篇英文”），系统会先按你的意图设计返回格式，再多源检索并整理成结构化文献清单（含引用与溯源）。
    输入方向后点击「检索」生成检索报告。

    写作助手

    面向研究人员的写作小工具：选择一个任务，按提示填入信息，Hy3 帮你完成对应的写作环节。涉及真实性（如引言、相关工作）的任务，请直接提供相关文献及其说明，Hy3 会据此进行真实引用。

    选择左侧任务并填写信息后点击「生成」。

    思路提炼

    与 Hy3 就你的研究想法进行互动式推演：厘清问题、锚定研究基点、提出可证伪假设，并基于真实文献给出证据支撑。 本模块不产出成稿，只帮你把思路想清楚；右侧「提炼指南」会把结论按已锚定与待论证两部分沉淀。

    💡 思路对话
    ▾
    描述你的研究兴趣或卡点，例如：「我想做医疗 RAG 的上下文压缩，但不知道从哪个点切入。」
    📚 检索依据
    ▾
    在推演中若判断某个论断需要用真实文献核验，会自主规划并检索依据，结果同步到右侧「本次引用」。
    🧭 提炼指南
    ▾
    在左侧对话中梳理思路后，点击「生成指南」导出一份结构化思路提炼结果。 指南会按已锚定的研究要素与待论证的研究要素两组沉淀，可作为下一步研究的决策底稿。
